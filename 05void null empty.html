Assignment Questions and Answers

  1. What are void tags in HTML?

    Void tags are HTML elements that do not have content between start and end tags; they are self-contained.

  2. Give three examples of void tags.

    Examples include <br>, <img>, and <hr>.

  3. What is the purpose of the <br> tag?

    The <br> tag inserts a line break in the text.

  4. How does the <img> tag work?

    The <img> tag displays an image and requires attributes like src and alt.

  5. What does the <hr> tag do?

    The <hr> tag creates a horizontal line to separate content.

  6. Can void tags have attributes?

    Yes, void tags can have attributes just like regular tags, such as src and alt in the <img> tag.

  7. Are void tags required to be closed with a trailing slash?

    No, in HTML5, void tags can be written with or without the trailing slash.

  8. What is the role of the <meta> tag?

    The <meta> tag provides metadata about the HTML document, such as character set and description.

  9. How is the <link> tag used?

    The <link> tag links external stylesheets to the HTML document.

  10. Why are void tags important in HTML?

    Void tags are important for inserting content like line breaks, images, and metadata without needing additional markup.
