Practical Use

The <a> tag is useful for:

  • Creating links to other web pages, both internal and external.
  • Linking to specific sections within a page using anchor links.
  • Facilitating email communication by linking to email addresses.
  • Providing downloadable files for users.

Assignment Questions and Answers

  1. What is the purpose of the <a> tag in HTML?

    The <a> tag is used to create hyperlinks, allowing users to navigate to other pages or resources.

  2. What is the href attribute in the <a> tag?

    The href attribute specifies the URL of the page or resource that the link points to.

  3. What does the target attribute do?

    The target attribute specifies where to open the linked document. Common values include _blank for a new tab and _self for the same frame.

  4. Can the <a> tag link to an email address?

    Yes, you can use the mailto: scheme in the href attribute to create a link that opens the user's email client.

  5. How do you create an anchor link within the same page?

    To create an anchor link, use the <a> tag with the href attribute set to the ID of the target element, preceded by a #.

  6. What is the difference between an external link and an internal link?

    An external link points to a different website, while an internal link navigates within the same website or page.

  7. How can you make a link open in a new tab?

    To open a link in a new tab, use the target="_blank" attribute in the <a> tag.

  8. Is it possible to style links with CSS?

    Yes, links can be styled using CSS, allowing you to change their color, font, size, and other properties.

  9. What happens if you do not include the href attribute in an <a> tag?

    If the href attribute is not included, the link will not navigate anywhere when clicked.

  10. Can you use the <a> tag to link to a file?

    Yes, you can link to files (such as PDFs or images) by specifying the file's path in the href attribute.

Definition and Usage

The <a> tag defines a hyperlink, which is used to link from one page to another.

The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

Tips and Notes

Tip: If the <a> tag has no href attribute, it is only a placeholder for a hyperlink.

Tip: A linked page is normally displayed in the current browser window, unless you specify another target.

Tip: Use CSS to style links: CSS Links and CSS Buttons.

Default CSS Settings

Most browsers will display the <a> element with the following default values:

a:link, a:visited {
  color: (internal value);
  text-decoration: underline;
  cursor: auto;
}

a:link:active, a:visited:active {
  color: (internal value);
}
