Absolute vs Relative Path in HTML

Absolute Path

An absolute path specifies the complete URL to the resource.

<img src="https://www.example.com/images/photo.jpg" alt="An example image">

Relative Path

A relative path specifies the location of the resource in relation to the current document's location.

<img src="images/photo.jpg" alt="An example image">

Directory Structure:

/project
    /images
        photo.jpg
    index.html

Practical Use

Understanding when to use absolute vs relative paths is crucial for:

  • Ensuring links to external resources work correctly regardless of the document's location.
  • Making it easier to manage resources within a single project.
  • Facilitating website maintenance and updates.

Assignment Questions and Answers

  1. What is an absolute path?

    An absolute path specifies the complete URL to a resource, including the protocol and domain.

  2. What is a relative path?

    A relative path specifies the location of a resource in relation to the current document's location.

  3. When should you use an absolute path?

    Use an absolute path when linking to resources hosted on different domains.

  4. When is a relative path preferred?

    A relative path is preferred when linking to resources within the same website or project.

  5. How does changing the directory structure affect relative paths?

    Changing the directory structure can break relative paths, whereas absolute paths remain valid as they include the full URL.

  6. What is the potential downside of using absolute paths?

    Absolute paths can make the code less portable; if the domain changes, all links must be updated.

  7. Can a relative path include `../`?

    Yes, using `../` allows you to navigate up one level in the directory structure.

  8. Can you mix absolute and relative paths in the same document?

    Yes, you can use both absolute and relative paths in the same HTML document as needed.

  9. How would you link to a CSS file using an absolute path?

    <link rel="stylesheet" href="https://www.example.com/styles.css">

  10. How would you link to a JavaScript file using a relative path?

    <script src="scripts/app.js"></script>
